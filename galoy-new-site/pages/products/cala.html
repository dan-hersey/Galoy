Cala

Core Accounting Ledger. An embeddable double-entry accounting system for managing multiple user accounts at scale.

Request Demo View on GitHub

The Foundation of Financial Software

Every financial application needs a ledger. Cala provides the double-entry accounting foundation that fintech companies and banks need, without building from scratch.

Ship Faster

Skip the complexity of building accounting infrastructure from scratch. Embed Cala and focus on your product.

Always Balanced

True double-entry accounting means your books always balance. No reconciliation nightmares.

Complete Audit Trail

Every transaction recorded with full history. Ready for regulators and auditors.

Enterprise-Grade Ledger Capabilities

Everything you need for sophisticated financial accounting

Multi-Currency Support

Native support for multiple currencies including fiat (USD, EUR, etc.) and cryptocurrencies (BTC, sats) in the same ledger.

Atomic Transactions

All-or-nothing transaction processing ensures consistency even in complex multi-account operations.

Flexible Account Hierarchy

Create any account structure you need: assets, liabilities, equity, income, expenses with unlimited nesting.

Real-Time Balances

Instant balance queries across any account, time period, or currency. No batch processing delays.

Immutable History

Append-only transaction log means nothing is ever deleted. Perfect for compliance and auditing.

GraphQL API

Modern GraphQL interface for querying accounts, transactions, and balances with precise data fetching.

Simple to Integrate

# Create accounts
mutation {
  createAccount(input: {
    name: "Customer USD Wallet"
    code: "customer-123-usd"
    normalBalanceType: DEBIT
  }) {
    accountId
  }
}

# Record a transaction
mutation {
  postTransaction(input: {
    transactionId: "txn-456"
    journalId: "main-journal"
    entries: [
      { accountId: "customer-123-usd", amount: 10000, currency: "USD", direction: DEBIT }
      { accountId: "revenue-fees", amount: 10000, currency: "USD", direction: CREDIT }
    ]
  }) {
    transactionId
  }
}
