About This Project

This is a speculative design exercise imagining what retail banking could look like in 2040 if Bitcoin becomes global financial infrastructure. The scenario assumes Bitcoin is a boring settlement layer — not an investment, not a movement, just plumbing.

Banks still exist. Fiat still exists. Custody is professional. The user never sees private keys. Everything just… works.
